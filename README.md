# packs-application

Frontend for the Packs Calculator using Vite + React + TypeScript and React Query.

## Getting started

```bash
pnpm i   # or npm i / yarn
pnpm dev # or npm run dev / yarn dev
```

Env:
- `VITE_PACKS_API_URL` (default `http://localhost:8082`)

Open http://localhost:5173 and calculate pack breakdowns for a quantity and optional custom sizes.
